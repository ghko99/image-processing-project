# Controls Reference

## Keyboard controls

- `Q`: quit the program.
- `R`: start or stop recording.
- `P`: play the latest recording.
- `C`: enter calibration mode.
- `S`: save settings.
- `H`: show help.

## Runtime files

- `config.json` stores local settings and is generated at runtime.
- `recordings/` stores local recording output.

Both paths are ignored by Git because they are machine-specific runtime artifacts.
